### Tooltips:
```
git commit --amend
git revert <SHA-of-commit-to-revert>
git reset <reference-to-commit>
git log [<options>]
    -p
    --stat
    --decorate
    --author="..."
    --grep="..."
    --pretty=format:"..."
git branch [<options>]
    --merged
    --no-merged
git shortlog [<options>]
    -s
    -n
```
